## 🔧 系统启动命令

### 一键启动系统
```bash
# 进入部署目录
cd quant-trading-system/deploy

# 授予执行权限
chmod +x start_system.sh

# 启动系统
./start_system.sh
```

### 手动启动（如果需要）
```bash
# 1. 构建Docker镜像
docker-compose build

# 2. 启动数据库服务
docker-compose up -d postgres redis

# 3. 应用数据库迁移
docker-compose run --rm backend python manage.py migrate

# 4. 启动所有服务
docker-compose up -d
```
