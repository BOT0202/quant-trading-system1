## 🔧 工具详解

### data_collector.py - 数据收集器
**功能**: 收集基础市场数据和策略示例
**使用场景**: 快速获取测试数据，了解数据结构

**主要功能**:
1. 收集加密货币市场数据（BTC, ETH, BNB）
2. 收集交易策略示例（5个经典策略）
3. 数据存储到SQLite数据库
4. 导出CSV格式数据
5. 生成数据收集报告

**使用示例**:
```python
from data_collector import DataCollector

# 初始化
collector = DataCollector("my_data.db")

# 收集数据
collector.collect_crypto_data()      # 市场数据
collector.collect_strategy_examples() # 策略数据

# 导出和分析
collector.export_to_csv()            # 导出CSV
collector.generate_report()          # 生成报告
```

### strategy_scraper.py - 策略爬取器
**功能**: 从网络自动爬取交易策略
**使用场景**: 大规模收集策略，进行筛选和分析

**数据来源**:
1. **量化社区**: QuantConnect、聚宽、JoinQuant等
2. **学术论文**: 金融学术期刊和论文
3. **GitHub**: 开源量化项目

**主要功能**:
1. 多源策略爬取
2. 策略分析和评分
3. 风险等级分类
4. 导出Excel和JSON格式

**使用示例**:
```python
from strategy_scraper import StrategyScraper

# 初始化
scraper = StrategyScraper("strategies.db")

# 爬取策略
scraper.scrape_community_strategies()   # 社区策略
scraper.scrape_academic_strategies()    # 学术策略
scraper.scrape_github_strategies()      # GitHub策略

# 分析策略
analysis = scraper.analyze_strategies()  # 分析报告

# 导出数据
scraper.export_strategy_list()          # 导出列表
```

## 📊 数据模型

### 策略表结构
```sql
CREATE TABLE strategies (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    name TEXT NOT NULL,              -- 策略名称
    category TEXT,                   -- 策略类别
    source TEXT,                     -- 数据来源
    description TEXT,                -- 策略描述
    performance_score REAL,          -- 性能评分(0-10)
    risk_level TEXT,                 -- 风险等级
    created_at TIMESTAMP,            -- 创建时间
    updated_at TIMESTAMP             -- 更新时间
);
```

### 市场数据表结构
```sql
CREATE TABLE market_data (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    symbol TEXT NOT NULL,            -- 交易对符号
    price REAL,                      -- 价格
    volume REAL,                     -- 交易量
    change_percent REAL,             -- 涨跌幅(%)
    timestamp TIMESTAMP              -- 时间戳
);
```

### 场景2: 按类别分析策略
```python
import sqlite3
import pandas as pd

conn = sqlite3.connect('strategies.db')

# 按类别统计
stats = pd.read_sql_query('''
SELECT category, 
       COUNT(*) as count,
       AVG(performance_score) as avg_score,
       MIN(performance_score) as min_score,
       MAX(performance_score) as max_score
FROM strategies
GROUP BY category
ORDER BY avg_score DESC
''', conn)

print("策略类别分析:")
print(stats.to_string())

# 保存分析结果
stats.to_csv('category_analysis.csv', index=False)
```

### 3. 数据质量检查
```python
# 检查数据完整性
def check_data_quality(db_path):
    conn = sqlite3.connect(db_path)
    
    checks = {
        '总策略数': "SELECT COUNT(*) FROM strategies",
        '有描述的策略': "SELECT COUNT(*) FROM strategies WHERE description IS NOT NULL",
        '有评分的策略': "SELECT COUNT(*) FROM strategies WHERE performance_score IS NOT NULL",
        '数据完整性': "SELECT COUNT(*) FROM strategies WHERE name IS NOT NULL AND category IS NOT NULL"
    }
    
    results = {}
    for check_name, query in checks.items():
        count = conn.execute(query).fetchone()[0]
        results[check_name] = count
    
    conn.close()
    return results

quality = check_data_quality('strategies.db')
for check, count in quality.items():
    print(f"{check}: {count}")
```

## 📈 性能优化

### 数据库优化
```sql
-- 创建索引提高查询性能
CREATE INDEX idx_strategies_score ON strategies(performance_score);
CREATE INDEX idx_strategies_category ON strategies(category);
CREATE INDEX idx_strategies_risk ON strategies(risk_level);
CREATE INDEX idx_market_symbol ON market_data(symbol);
CREATE INDEX idx_market_timestamp ON market_data(timestamp);

-- 定期清理旧数据
DELETE FROM market_data WHERE timestamp < DATE('now', '-30 days');
VACUUM;
```

### 爬取优化
```python
# 使用连接池
import requests
from requests.adapters import HTTPAdapter
from urllib3.util.retry import Retry

session = requests.Session()
retry = Retry(total=3, backoff_factor=0.1)
adapter = HTTPAdapter(max_retries=retry, pool_connections=10, pool_maxsize=10)
session.mount('http://', adapter)
session.mount('https://', adapter)
```
