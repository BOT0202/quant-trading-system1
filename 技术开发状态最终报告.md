   - 单个仓位 ≤ 20% 总资金

   - 最小仓位 ≥ 1% 总资金

2. **回撤控制规则**
   - 日最大亏损 ≤ 5%
   - 总最大回撤 ≤ 10%
   - 软止损: 8%，硬止损: 10%

3. **波动率限制规则**
   - 日最大波动率 ≤ 3%
   - 仓位最大波动率 ≤ 5%
   - 基于ATR的动态仓位调整

4. **集中度控制规则**
   - 最大资产集中度 ≤ 25%
   - 最大行业集中度 ≤ 40%
   - 最大相关性 ≤ 70%

5. **流动性管理规则**
   - 最小流动性比率 ≥ 10%
   - 最大滑点 ≤ 0.5%
   - 最小成交量比率 ≥ 1%
